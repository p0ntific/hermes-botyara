# Hermes Botyara 🚀

AI-агент (Telegram Userbot) для автоматического поиска и квалификации B2B лидов в Telegram-чатах с помощью YandexGPT.

## Функционал
1. Сканирует целевые чаты (супергруппы).
2. Анализирует каждое сообщение через LLM на релевантность.
3. Отсеивает спам, саморекламу и ботов (фильтр `SKIP`).
4. Формирует персонализированный питч (с паузами от 30 до 180 сек для имитации человека).
5. Ведет диалог в личных сообщениях до выявления интереса.
6. Отправляет теплых лидов менеджеру и шлет отчеты в отдельную группу.

## Быстрый старт (через Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/p0ntific/hermes-botyara.git
   cd hermes-botyara
   ```
2. Скопируйте файл конфигурации и заполните свои данные:
   ```bash
   cp .env.example .env
   nano .env
   ```
3. Запустите бота:
   ```bash
   docker-compose up -d
   ```

## Настройка GitHub Actions для деплоя
В репозитории настроен автоматический деплой на ваш сервер при пуше в ветку `main`.
Для этого добавьте в настройки репозитория на GitHub (Settings -> Secrets and variables -> Actions) следующие секреты:
- `SERVER_HOST`: IP адрес вашего сервера
- `SERVER_USER`: Имя пользователя SSH
- `SERVER_SSH_KEY`: Приватный SSH ключ (Ed25519 или RSA)

*Скрипт деплоя ожидает, что проект на сервере лежит по пути `~/.hermes` и запускается через `systemctl --user restart tg_lead_skill.service`, но это можно адаптировать под docker в `.github/workflows/deploy.yml`.*
